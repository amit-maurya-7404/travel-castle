🚀 QUICK START (4 STEPS):

STEP 1: MongoDB Setup (5 minutes)
  → Go to mongodb.com/cloud/atlas
  → Create free account & cluster
  → Create database user
  → Get connection string (save it!)

STEP 2: Add Environment Variables (2 minutes)
  → Go to Vercel project settings
  → Add MONGODB_URI (your connection string)
  → Add JWT_SECRET (random secure string)
  → Save

STEP 3: Deploy (1 click)
  → Push changes to GitHub
  → Vercel auto-deploys
  → Wait for "Ready" status

STEP 4: Create Admin Account (1 minute)
  → Go to https://yourdomain.com/admin/login
  → Click "Register"
  → Create your admin account
  → Login to dashboard!
